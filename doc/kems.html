initiate/1

initiate(Remote) -> Result

  • Remote = axolotl:remote()
  • Result = #key_exchange_message{}

Generate key exchange message data. A key exchange mesage (KEM) has a randomly generated id which is used to match the id in the response from the other peer. This id is chosen randomly to make it unpredictable. Also, a KEM contains a public handshake (base) key and public initial ratchet key. These keys are in the specs refered to as A0/B0 and A1/B1 resp. If this function is called repeatedly, the old KEM is deleted and a new one is issued.
